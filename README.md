## TurboMarket Web

Plataforma web desarrollada con HTML, CSS y JavaScript para conectar comercios locales de Turbo — restaurantes, farmacias, licoreras y tiendas — con la comunidad mediante pedidos y servicios digitales en una sola plataforma.

Objetivo

Este proyecto busca impulsar la digitalización del comercio local en Turbo, ofreciendo una solución web accesible donde los usuarios puedan encontrar negocios cercanos, consultar productos y realizar pedidos fácilmente.

Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

Características

* Visualización de comercios locales
* Interfaz amigable y responsive
* Catálogo de productos y servicios
* Plataforma centralizada para negocios del municipio

Enfoque del proyecto

La plataforma está diseñada especialmente para zonas donde actualmente no existen aplicaciones o servicios digitales similares, fortaleciendo la conexión entre clientes y negocios locales.
